Make Figures Great Again (MFGA) is the best way to make plots. Period. With nine terrific color palettes, and Python and Adobe modules, MFGA is hands down—and look, we know something about hands here folks—the most incredible way to elevate the class and impact of your figures. American scientific figures used to be great. Make them great again.

Installation and Use

Python installation

pip install make-figures-great-again

Python usage example

# Make a plot with the "neck" color palette (top of this webpage)
import matplotlib.pyplot as plt
import make_figures_great_again
fig, (ax_left, ax_right) = plt.subplots(1, 2, figsize=(6,3))
fractions = [0.5,0.05,0.15,0.05,0.15,0.1,]

# Import some terrific colors
terrific_colors = make_figures_great_again.the_best_colors.neck_6.mpl_colors

# Default (Ugly, just awful)
ax_left.pie(fractions)  

# The greatest, terrific colors
ax_right.pie(fractions, colors=terrific_colors,)

plt.show()
